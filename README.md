# Ứng dụng Quản lý công việc - nguyenbaolam_2205httb029

Ứng dụng đơn giản để quản lý công việc cá nhân, sử dụng PHP và MySQL.

## Hướng dẫn cài đặt
1. Tạo database MySQL tên `quanlycongviec` và import file sql/database.sql.
2. Chỉnh sửa config/db.php với thông tin DB của bạn (username, password).
3. Chạy server local (XAMPP/WAMP/MAMP) và truy cập http://localhost/public/login.php để đăng ký/đăng nhập.
4. Chức năng: Đăng ký/đăng nhập/đăng xuất, thêm/sửa/xóa công việc, đánh dấu hoàn thành, hiển thị danh sách công việc cá nhân.
